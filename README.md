# DB Corpora

Database for collecting and refining corpus data. SQL database with web interface for users and API for direct SQL requests.

## Enabling Git Hooks

Remove hook samples and add symlink:

```bash
  rm -rf .git/hooks && ln -s ../hooks .git/hooks`
```
